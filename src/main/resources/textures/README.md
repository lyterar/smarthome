# Текстуры комнат

Папка зарезервирована для внешних текстур (.png, .jpg).

Текущая реализация использует **процедурные текстуры** (WritableImage),
генерируемые в `RoomTextureFactory.java` без внешних файлов.

## Как добавить свои текстуры

Если нужны реальные фото-текстуры — поместите PNG сюда и загрузите через:

```java
Image tex = new Image(getClass().getResourceAsStream("/textures/wood.png"));
mat.setDiffuseMap(tex);
```
