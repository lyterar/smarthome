# 3D Модели устройств умного дома

Папка для OBJ-файлов устройств. Файлы автоматически загружаются при старте.
Если файл отсутствует — используется геометрический примитив.

## Соответствие файлов типам устройств

| Файл              | Тип устройства | Описание              |
|-------------------|----------------|-----------------------|
| `light.obj`       | LIGHT          | Умная лампа           |
| `thermostat.obj`  | THERMOSTAT     | Термостат             |
| `sensor.obj`      | SENSOR         | Датчик движения/тепла |
| `lock.obj`        | LOCK           | Умный замок           |
| `camera.obj`      | CAMERA         | IP-камера             |
| `speaker.obj`     | SPEAKER        | Умная колонка         |

## Где скачать бесплатные OBJ-модели

- **Sketchfab** (фильтр: Free + OBJ): https://sketchfab.com/features/download
- **TurboSquid Free**: https://www.turbosquid.com/Search/3D-Models/free/obj
- **Free3D**: https://free3d.com/3d-models/obj
- **Clara.io**: https://clara.io/library

## Требования к OBJ-файлам

- Формат: стандартный Wavefront OBJ (`.obj`)
- Поддерживаются: вершины `v`, текстуры `vt`, нормали `vn`, грани `f`
- Грани: треугольники или квады (автоматическая триангуляция)
- Размер: ~10–40 единиц по высоте (координатная система JavaFX 3D)
- MTL-файлы игнорируются — цвет задаётся программно из состояния устройства

## Масштаб

По умолчанию scale=1.0. Если модель слишком большая/маленькая —
можно изменить в `ObjModelLoader.load(modelName, scale)` вызове из Device3DModel.
